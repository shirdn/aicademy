# _example Module

> Reference module — copy this to start a new frontend module.

## Purpose

This is a template module showing the standard structure. Copy it:

```bash
cp -r frontend/modules/_example frontend/modules/my-feature
```

Then rename files and update the README.

## Structure

```
_example/
├── README.md              # This file — describe your module here
├── src/
│   ├── components/        # UI components
│   ├── hooks.*            # Custom hooks (React) or composables (Vue)
│   ├── utils.*            # Module utilities
│   └── index.*            # Public exports
└── tests/
    ├── unit/              # Component + logic tests
    └── integration/       # Interaction / page-level tests
```
